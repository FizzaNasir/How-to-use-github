Difference Between git and github

Git is a distributed version control tool that can manage a development project's source code history

GitHub is a cloud based platform built around the Git tool

Github is a platform where you can put your code and can mainatain your repository. It is every effective to use github when you are working as a team.

Github in team effectiveness
