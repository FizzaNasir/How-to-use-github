Difference Between git and github

Git is a distributed version control tool that can manage a development project's source code history

GitHub is a cloud based platform built around the Git tool

Github is a platform where you can put your code and can mainatain your repository. It is every effective to use github when you are working as a team.

Github in team effectiveness

When you are working in a team, the changes made in the file by one member can be accessed to other members as well just by using 1 command(will be described next)

Git commands

git clone

It is used to clone a remote git repository to your local machine. Once you clone a repository, now any changes you would make in your local machine folder, will reflect in git repo (using commands describe next )

git add .

It is used to add files to your remote repository which have added or modified

git commit -m 'any msg here'

Commit saves the changes you made only to the local repository but not to the remote repository. In the message string, you can write the changes you made or the file you added so that everyone could see what have you pushed.
